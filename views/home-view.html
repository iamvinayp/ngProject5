An API Of Ice And Fire

This is an advanced angularJs single page web application which is about a popular TV Series Game of Thrones and its related book series. It is an SPA containing few views which includes

View 1

A view to list all books, characters and houses.

View 2

A view containing the detail of a particular book/character/house containing all the information about it in details.
